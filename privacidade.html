Política de Privacidade

icarsp.com.br — última atualização: março de 2026

1. Coleta de dados

Os dados de placa informados neste site são utilizados exclusivamente para consulta nas bases de dados públicas disponíveis, com o objetivo de gerar o relatório veicular contratado pelo usuário.

2. Uso dos dados

Não compartilhamos, vendemos nem armazenamos dados pessoais de terceiros. As informações fornecidas pelo usuário (nome, e-mail, telefone) são usadas apenas para comunicação sobre o serviço contratado.

3. Pagamentos

Os pagamentos são processados por gateways seguros (EFI Bank / Mercado Pago). Não armazenamos dados de cartão de crédito.

4. Cookies

Este site utiliza cookies de análise (Google Analytics) para entender o comportamento dos visitantes e melhorar o serviço. Nenhum dado pessoal identificável é coletado por meio de cookies.
